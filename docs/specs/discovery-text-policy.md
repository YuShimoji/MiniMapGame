# Discovery Text 設計方針

status: done
related: SP-020

本文書は Discovery テキスト60件の設計・執筆方針を定義する。
SP-020 (探索体験ポリッシュ) Layer 1 のコンテンツ方針として参照される。

---

## 1. 基本原則

### 排除するもの

| 排除対象 | 具体例 |
|---------|--------|
| 固有名詞 | 人名、地名、組織名、計画名 |
| 人間の存在 | 生活痕跡、仄めかし、行動の痕跡 |
| 陰謀・ミステリー | 秘密組織、隠蔽、調査、事件 |
| 主観・身体的知覚 | 温い、冷たい、鮮やか、匂い、肌触り |
| 時間軸 | 古い/新しい、最近、何年も、歳月、季節、時代 |
| 能動的観察行為 | 測る、数える、時間経過の追跡 |

### 描写するもの

**空間・環境の客観的断片のみ。**

光の入り方、空気の挙動、壁面の質感、構造の不整合、
物質の状態変化(結露、錆、堆積)を淡々と記述する。

---

## 2. 視点と文体

### 視点

- **客観描写のみ**。主語を置かない。
- 観察者の存在を消す。「～と感じる」「～に見える」を避ける。
- ただし「見えない」「判然としない」など、空間の性質として
  不可視性を述べることは許容する。

### 文体

- **自然な散文**。「～ている。～ている。」のぶつ切り・無骨な羅列を避ける。
- **ジュリアン・グラック、ガルシア・マルケスを参照**:
  精密で淡々としながら、異質なものを当然のように述べる語り口。
- 文は流れを持ち、従属節で精度を重ねる。

### 「気付き」の付加

客観描写の後に、**なぜその空間が目を引くのか**を示す
短い観察・事実をさりげなく添える。

**良い例** (即時に視認できる空間的事実):
- 「配管が奥の壁へ消えている。手前と奥で本数が一致しない」
- 「ガラスパネルの内側にだけ結露が生じ、外側は乾いたままでいる」
- 「天井タイルが数枚抜け落ちている。抜けた箇所は等間隔に並んでいる」

**避けるべき例**:
- 時間経過の観察: 「消灯の瞬間だけ天井の高さが変わる」
- 能動的測定: 「どこを測っても同じ」
- 因果の断絶: 「放射状の罅。その中心点には何の痕跡もない」(罅と掛からない)
- 物理的不可能の乱用: 「煉瓦だが光が透過」(材質の世界観に波及する)
- ニッチすぎる違和感: 「双方とも水平だが互いの水平が一致しない」(単なる段差に読める)

### 気付きの基準

即時に、受動的に、その場で視認できる空間的事実であること。
時間の前後比較、能動的な計測、不可視の因果推論を含まない。

## 3. カテゴリ別の空間質

建物カテゴリによって空間の物理的性質が変わる。
人間的な用途やテーマではなく、**空間そのものの質**で差異を出す。

| BuildingCategory | 空間質 | キーワード |
|-----------------|--------|-----------|
| Residential | 狭い、柔らかい素材、低い天井 | 漆喰、壁紙、絨毯、磨りガラス |
| Commercial | 開けた、硬い表面、人工照明 | リノリウム、蛍光灯、ガラス、金属棚 |
| Industrial | 広大、金属的、反響 | 波板、鉄骨、コンクリート、配管 |
| Public | 反響、石材、高い天井 | 大理石、アーチ、モールディング、花崗岩 |
| Special | 異質、予想外、不整合 | 素材の不一致、幾何の歪み、説明できない現象 |
| ANY (null) | 特定の質を持たない | 埃、光、隙間風、塗料、配管 |

---

## 4. レアリティと解像度

レアリティは**描写の解像度**(粒度と分量)を決定する。
世界観やトーンは変えない。

| Rarity | 解像度 | 分量目安 |
|--------|--------|---------|
| Common (70%) | 粗い観察。空間のひとつの特徴を捉える | 1文 |
| Uncommon (25%) | やや詳細。特徴に加え、空間的な不整合や気付きを添える | 1-2文 |
| Rare (5%) | 細密。空間の構成を複数の要素で精密に描写する | 2-3文 |

---

## 5. furnitureType の扱い

コード上の `furnitureType` (Document / Note / Photo / Container) は
**出現率バランスと内部分類のためにのみ維持**する。

テキスト内容には furnitureType を反映しない。
全タイプで同一の空間描写方針を適用する。

---

## 6. 言語

- EN / JA 両フィールドを維持する。
- 日本語で執筆し、英語は同等の品質で対訳する。
- 両言語とも同一の方針に従う。

---

## 7. 禁止パターン一覧

執筆・レビュー時のチェックリストとして使用する。

- [ ] 固有名詞が含まれていないか
- [ ] 人間の存在や行動を仄めかしていないか
- [ ] 主観的な知覚表現(温度、触覚、嗅覚)がないか
- [ ] 明示的な時間軸(古い/新しい、何年、最近、季節)がないか
- [ ] 能動的な観察行為(測る、数える、比較する)を前提としていないか
- [ ] ぶつ切りの無骨な文体になっていないか
- [ ] 気付きの部分が即時・受動・視認可能な事実であるか
- [ ] カテゴリの空間質と合致しているか
- [ ] レアリティに応じた解像度(分量)であるか
